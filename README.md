# opz-key-finder

[Live demo](https://yifanwu.studio/key-finder/)

## About

Provide key reference while jamming.

Features:

- OP-Z inspired click component
- Interactive with MIDI inputs via Web MIDI API
- Drag and drop MIDI files

## Building

```sh
yarn install
yarn start
```
